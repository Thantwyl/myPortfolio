# My Portfolio

A personal portfolio website built with React and Vite. It showcases projects, skills, and contact links.

## Features
- Modern React + Vite setup with fast HMR
- Responsive layout and dark/light styles
- Framer Motion animations
- Project list, skill sections, and contact links

## Credits
- Built with [Vite](https://vitejs.dev/) and [React](https://reactjs.org/)
- Animations with [Framer Motion](https://www.framer.com/motion/)
- Icons from `react-icons`
